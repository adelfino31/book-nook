# 📚 The Book Nook

A cozy reading tracker designed with an old-library aesthetic.

## Features

- 📖 Track your current read
- ⭐ Rate finished books
- ✍️ Write short reviews
- 📚 Virtual bookshelf view
- 💾 Local storage persistence

## Technologies

- HTML
- CSS
- JavaScript
